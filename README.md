# 简介
Corelink 后台 nest.js 工具

# 安装
```
npm i -g @corelink/backend-cli
```

# 使用方法

> 帮助
```
cbc -h
```

> 新建项目
```
cbc -n
```

> 新建API

```
cbc -a
```
